Mi Portafolio

Bicitour X

Plataforma diseñada para conectar ciclistas y promover rutas interactivas. Incluye registro de usuarios, gestión de eventos y mapas en tiempo real. Desarrollado con Angular en el frontend y Node.js con Express en el backend.

E-commerce UI

Diseño y desarrollo de una tienda online con carrito de compras, filtros avanzados y pagos en línea. Implementado con React, Bootstrap, y Firebase para la gestión de productos y autenticación de usuarios.

Dashboard Analytics

Panel de análisis de datos interactivo que permite visualizar métricas en tiempo real. Construido con Angular y gráficos dinámicos usando Chart.js, integrado con Firebase Firestore para la recolección de datos en tiempo real.

Task Manager App

Aplicación web para la gestión de tareas y productividad. Permite crear, editar y eliminar tareas con notificaciones automáticas. Desarrollado con Vue.js , Node.js y MongoDB para almacenamiento de datos.

Blog Personal

Blog interactivo con sistema de comentarios y categorías. Utiliza Next.js para un rendimiento optimizado y Markdown para la gestión de contenido. Base de datos en Firebase Firestore y autenticación con Google Auth .
